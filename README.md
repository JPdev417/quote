# quote
Simple program to get a random quote from a list


## Building

```bash
gcc quote.c -o quote
```

---

## Usage

1. Create a `quotes.txt` file in your `~/.config/` directory with one quote per line.

2. Run the program:

```bash
./quote
```

Each run prints a random colored quote from your list.
